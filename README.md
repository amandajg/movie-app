# Movie App

This React application utilizes the The Movie Database (TMDB) API to populate a page with popular movies and allows the user to search the database for movies by title. The user is also able to mark movies as a "favorite", which are all shown on a separate page.

## TMDB API

This API provides data to this application such as movie title, release date, poster, and more. An API key is needed to fully run this project. To learn more about this API and how to register for an API key, please check out the [Getting Started](https://developer.themoviedb.org/docs/getting-started) page.
